# HTML

시작은 항상 <!DOCTYPE html>로 시작
<html lang="ko"> lang = 언어, 명시해 주는 것이 좋다
    
<head>
    <meta charset="utf-8" /> = 인코딩
    <title>HTML 시작하기</title> = 웹 문서 제목
    <style>
        body {
            background-color: lightblue;
        }
    </style>
</head>
    
<body>
    <h1>웹 개발 기초</h1> = 제목?
    <p>HTML</p> p = 단락
    <p>CSS</p>
    <p>JavaScript</p>
</body>

</html>
